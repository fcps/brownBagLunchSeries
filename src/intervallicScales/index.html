A Bit of Math

  1. $a + b = b + a$
  2. If $a = c$ AND $b = d$ THEN $a + b = c + d$
  3. $(a + b) + c = a + (b + c)$


a ⊕ b ⊜ b ⊕ a

  • Unlike the Ordinal Scale, combination rules exist on the intervallic scale
  • This first rule is the Commutative Property of Addition
  • We can sum the values of two variables measured on an intervallic scale in any order to get the same result


If a ⊜ c AND b ⊜ d
THEN a ⊕ b ⊜ c ⊕ d

  • This is the Transitive Property of Equality
  • If variables are equivalent combining those variables will also be equivalent
  • This property is also allows us to define subtraction

(a ⊕ b) ⊕ c ⊜ a ⊕ (b ⊕ c)

  • This is known as the Associative Property
  • The grouping will not affect the sum of the values if the variables appear in the same order

What is an intervallic scale

  • It is a measurement scale that does not have a defined true zero
  • Some assessments are capable of producing intervallic scales (e.g., NWEA MAP)
  • The location of 0 is arbitrary (e.g., can mean something other than no quantity)
  • How about an example?
  • Explain arbitrary zero
  • Some times zero does not exist in the scale like the weight of humans
  • Other times the value of zero has a different meaning than a lack of the quantity
