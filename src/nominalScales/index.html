A Bit of Math

  1. Either $a = b$ OR $a \neq b$
  2. If $a = b$ then $b = a$
  3. If $a = b$ & $b = c$ then $a = c$

Either a ⊜ b or not


  • This means we can test whether or not a value is equal to another value.
  • It also means that values in a nominal scale are mutually exclusive.

If a ⊜ b then b ⊜ a

  • The notational change here is related to set theory.
  • Basically if there are two categories of a group this means we can map any number to the category and it will not convey any meaning

If a ⊜ b and b ⊜ c then a ⊜ c

  • This generalizes the previous rule to cases where there are more than two groups

What is a nominal scale

  • A mapping of numeric values to categories of like groups
  • Because stats and math relies on numbers for computation we need a way to represent groups numerically
  • For a nominal scale, the numeric values only serve to identify a specific category

  • BMW, Ford, GM, and Volvo could all be included in a nominal scale because they are all automobile manufacturers (e.g., like objects)
  • Since bananas and cats are not auto manufacturers they are not comparable and could not be included in the same scale
  • As long as the data in the nominal scale can be meaningfully grouped into a single concept it will work to allow you to identify those categories
  • Nominal scales are either used to identify a category without any natural ordering to it
  • Or they are used in the dichotomous was to indicate correct/incorrect values or something analogous

Other Information

  • Some nominal scales only have two values (e.g., Male/Female, Correct/Incorrect, etc...)
  • These variables are sometimes called binary or dichotomous (e.g., 0/1) variables
  • These values can be transformed using any 1:1 mapping (e.g., the numbers only need to identify unique categories)
