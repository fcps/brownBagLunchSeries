What is an ordinal scale

  • Ordinal scales are used order values based on magnitude.
  • Rankings serve a similar purpose but the direction of the numbers is reversed.
  • If you've ever used a grading rubric, you're already familiar with how ordinal measurement scales are used.
  • Rubrics are a great example, but in some cases the rubric design may add additional problems to the measurement
  • One major issue with rubric design is avoiding multidimensional criterion
  • In other words, a higher score for an indicator should only depend on an increase in magnitude in a single dimension.

  • Like Nominal scales, ordinal scales must also represent a single dimension

Ordinal Scales and Rankings


  • Transformations between rankings and ordinal scales are fairly easy.
  • The difference between an ordinal scale and a ranking is the interpretation of the values.
  • In a ranking a lower number indicates a greater amount of the quantity being measured.
  • In an ordinal scale a higher number indicates a greater amount of the quantity being measured.
  • Here's our data $x = [1, 4, 2, 3]$
    1. $\alpha = x_{\max}$
    2. $(\alpha + 1) - x_{i}$
    3. $x' = [(4 + 1) - 1, (4 + 1) - 4, (4 + 1) - 2, (4 + 1) - 3]$
    4. $x' = [4, 1, 3, 2]$
  • This reverses the order of the integers to transform between ordinal and rankings
  • This assumes that the highest observed value is the highest possible value for the scale

Other Information

  • Binary/Dichotmous data can be considered a special case of ordinal data with only two values.
  • You can transform the values pretty easily as long as you preserve the order relationship.
