A Bit of Math

  1. $a + 0 = a$
  2. If $a = c$ AND $b \gt 0$ THEN $a + b \gt c$

a ⊕ 0 ⊜ a


  • Unlike the Intervallic Scale, the ratio scale has a true zero
  • This is known as the Additive Identity
  • This means that the value of zero is fixed and has a single meaning (e.g., the lack of a quantity)

If a ⊜ c AND b ⧁ 0 THEN a ⊕ b ⧁ c

  • This property constrains the distance between the integers to be equal by fixing the location of 0
  • The converse would also be true (e.g., if b < 0 then a + b < c

What is a ratio scale

  • This is what you would typically think of when using numbers in everyday situations
